# C++ Core Programming Showcase 🚀
<p>A collection of mini-systems and utility programs written in C++ demonstrating advanced syntax, pointer structures, modular programming, and Object-Oriented principles.</p>

---

### 📂 Featured Programs

#### 1. Student Academic Performance & Eligibility Tracker (`team15.cpp`)
- **Description:** A robust mini-system designed to manage, compute, and track student grades, marks, and eligibility criteria based on performance logic.
- **Key Features:** Data structures, conditional evaluation, and performance monitoring.

#### 2. Simple Calculator (`calculator.cpp`)
- **Description:** A console-based application to perform fundamental arithmetic operations (Addition, Subtraction, Multiplication, Division).
- **Key Features:** Error checking (e.g., division by zero) and switch-case logic.

#### 3. Age Checker & Utilities (`age_check.cpp` & `temp_conv.cpp`)
- **Description:** Handy utility scripts focusing on conditional flows (Age verification) and mathematical formulas (Celsius/Fahrenheit Temperature Conversion).
